Ajuste de texto (CSS)

nome função exemplo
text-align: alinhamento do texto justify; center; right;
text-transform: transforma uma caracteristica no texto inteiro uppercase; lowercase; capitalize;
text-ident: coloca uma identação no inicio do texto 35px
text-decoration: decoração do texto underline; overline; line-through; none;
letter-spacing: espaço entre as letras 1.2px; -2px; (padrão é 0)
line-height: altura da linha 2; 300%; 30px;
word-spacing: espaço entre as palavras 5px
text-shadow: sombra no texto 2px(dislocamento para direita) 2px(dislocamento para baixo) 5px(raio de desfoque) #bbb;(cor)
font-weight: grossura das letras 400;(bold)


fontes de texto (CSS)

existe 3 familias genericas de fontes: as com serifa(com detalhes nos cantos das letras), sem serifa e monospace(todas as letras com a mesma largura)

nome função exemplo
font-family: qual a fonte do texto arial, "comic sans MS", (causo a primeira não esteja disponivel a segunda é usada, no caso de fontes com mais palavras no nome é nescessario aspas)
font-style: estilo da fonte normal; italic;
font-variant: tudo na forma maiscula mas com variação normal no tamanho normal; small-caps;


no google fontes é possivel pegar outras fontes e o site font squirrel tranforma fontes em Woff(arquivo mais leve) e ja escreve o arquivo CSS para ela



para fontes personalizadas usar:
